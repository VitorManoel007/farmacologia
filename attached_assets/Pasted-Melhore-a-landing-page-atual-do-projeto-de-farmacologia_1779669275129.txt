Aumentar conversão emocionalmente.

A página deve vender:

* alívio
* revisão rápida
* facilidade
* memorização prática
* menos sofrimento estudando farmacologia.

IMPORTANTE:
Essa NÃO deve parecer:

* plataforma premium complexa
* software futurista
* startup tecnológica
* sistema complicado.

A página deve parecer:

* simples
* prática
* rápida
* útil
* criada para estudantes reais.

SENSAÇÃO PRINCIPAL:
“Finalmente achei um jeito mais fácil de revisar farmacologia.”
